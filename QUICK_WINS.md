# Quick Wins (Low Effort, High Impact)

## 1) Add "Stuck Items" banner on dashboard
Show count of items unchanged for 3+ days in active stages.

## 2) Add one-click "Mark Received" from search results
Reduce taps for scanner fallback name-search flow.

## 3) Add required confirmation for delete actions
Use typed confirmation (`DELETE`) for item/user removal.

## 4) Add item-location autocomplete
Suggest previously used shelf/bin values to reduce typos.

## 5) Add "Recent Scans" widget
Display last 10 scans with employee and time.

## 6) Add stage aging badges
Show hours/days in current stage directly on item cards.

## 7) Add duplicate-name warning within consigner session
Flag likely duplicates while still allowing override.

## 8) Add offline queue indicator
When offline, show queued actions count and last sync time.

## 9) Add printable intake summary sheet
Include consigner, item count, lot codes, and labels generated.

## 10) Add simple CSV import for bulk intake
Allow office prep uploads for pre-registered incoming lots.

## 11) Add role-aware homepage tiles
Hide irrelevant actions by role to reduce accidental taps.

## 12) Add per-item "last touched by" chip in list view
Improve ownership visibility without opening item detail.
