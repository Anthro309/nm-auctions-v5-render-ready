# Other Product Ideas (Non-AI, Workflow + Inventory Focus)

## Operations UX
1. **Stage Kanban board** with drag/drop between workflow stages.
2. **Quick-action keyboard mode** for desktop stations (scan, advance stage, add note, mark repair).
3. **Bulk edit panel** for category/condition/location updates across selected items.
4. **Saved mobile shortcuts** (one-tap buttons per employee role).

## Inventory Control
5. **Bin/shelf mapping** with barcode labels for physical locations.
6. **Chain-of-custody timeline** with mandatory reason codes for stage changes.
7. **Cycle-count mode** to audit inventory by aisle/shelf weekly.
8. **Duplicate code guardrails** (prevent conflicting lot numbers and consigner code collisions).

## Intake Quality
9. **Required field templates by category** (e.g., dimensions for furniture, material for decor).
10. **Condition checklist wizard** with standardized vocabulary.
11. **Photo count minimums** per stage before allowing advance.
12. **Intake session lock/finalize** to prevent accidental edits after labels are printed.

## Team Management
13. **Shift handoff summary** (what changed, blocked items, urgent items).
14. **Workload balancing view** by employee and stage.
15. **Task ownership assignment** per item for accountability.
16. **Audit export** (CSV/PDF) of activity logs for internal reviews.

## Reliability & Admin
17. **Automated nightly backup job** with restore UI.
18. **Change history restore point** per item (undo last edit).
19. **System health page** (storage size, last backup time, API errors).
20. **Feature flags panel** to safely roll out new workflow tools.
